Background

It was written with the aim of submitting it as an opinion paper to the special number of Biolinguistics on the prospects for the XXI century.

It is a philosophical paper, but mainly dealing with linguistics, so it can be read as an introduction to the linguistics papers.

Abstract

I defend that the main task of language is to deal with intended meanings, and therefore that language should be centered on semantics and pragmatics. I propose a subjectivist program based on problem solving to achieve that conception of language. I argue that the predominant program of language, which is centered on syntax, is driving biolinguistics to a dead-end.
